Apa itu Jurusan Pengembangan Perangkat Lunak dan Gim? (PPLG)

PPLG adalah jurusan singkatan Pengembangan Perangkat Lunak dan Gimd dan merupakan sebuah jurusan yang ada di Sekolah Mengengah Kejuruan (SMK).PPLG adalah sebuah jurusan yang mempelajari dan mendalami semua cara-cara pengembangan perangkat lunak termasuk pembuatan,pemeliharaan,manajemen organisasi pengembangan peerangkat lunak dan manajemen kualitas

Apa saja Yang Dipelajari Di Jurusan PPLG?

Sebetulnya PPLG (Pengembangan Perangkat Lunak Dan Gim) itu Gampang! tidak jauh sama 3 hal ini yaitu:

  1. Coding
  2. Desain
  3. Alogaritma

Didalam Coding,Desain dan Alogaritma tersebut masih ada Point2 penting yang dipelajari di Jurusan PPLG (Pengembangan Perangkat Lunak dan Gim)

Jurusan PPLG Kerja Apa?

Banyak yang tanya jususan PPLG itu nanti kalau lulus bisa kerja apa?apa saja lowongan kerja lulusan SMK jurusan PPLG. Berikut daftar peluang kerja jurusan PPLG:

  • Developer iT
  • Progammer
  • Web engineer
  • Game Developer
  • Database engineer / data base administrator
Bahasa pemrograman yang dipelajari:
  • Pemrogaman Bahasa C
  • Pemrogaman Bahasa C++

  • Pemrogaman Bahasa
